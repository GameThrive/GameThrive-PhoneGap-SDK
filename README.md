GameThrive PhoneGap Push Notification Plugin
====================================

GameThrive is a free push notification service for mobile apps. This plugin makes it easy to integrate your PhoneGap CLI, PhoneGap Build, Cordova, or Sencha Touch app or game with GameThrive. Supports Android, iOS, and Amazon's Fire OS platforms.

- See http://documentation.gamethrive.com/v1.0/docs/phonegap-sdk-overview for setup documentation.
